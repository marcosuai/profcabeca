Informação sobre o Projeto

Este projeto foi desenvolvido para alunos do ensino médio como uma introdução prática e didática ao desenvolvimento web utilizando HTML, CSS e JavaScript. O objetivo é proporcionar uma experiência hands-on, onde os alunos aprendem a criar um carrossel de imagens interativo e responsivo, além de explorar técnicas de animação e organização de conteúdo.

O layout foi projetado para ser visualmente atrativo e funcional, proporcionando uma experiência de aprendizado envolvente. Cada página, incluindo a página inicial, contato, informação, sobre e FAQ, foi criada com o mesmo estilo visual consistente, facilitando a navegação e a compreensão do conteúdo para os alunos.

Mais Informação sobre o Projeto

O HTML foi utilizado para estruturar o conteúdo das páginas, definindo os elementos que compõem o carrossel de imagens e os estilos básicos.

O CSS foi empregado para estilizar o projeto, definindo cores, fontes, layout responsivo e animações que tornam o carrossel visualmente atraente e funcional.

Já o JavaScript foi utilizado para criar a lógica interativa do carrossel, permitindo a navegação entre as imagens, a animação dos elementos e o controle automático do avanço das imagens.

Cada página, incluindo a página inicial, contato, informação, sobre e FAQ, foi criada com o mesmo estilo visual consistente, facilitando a navegação e a compreensão do conteúdo para os alunos.
